Basic button and active state:

```jsx
    <div> 
        <Button>Push Me</Button>
        <div style={{margin: '0px'}} />  
        <Button active>Push Me</Button>
    </div> 
```

Button with border:

```jsx
    <Button border>Push Me</Button>
```

Button with Icon:

```jsx
    <div> 
        <Button icon></Button>
        <div style={{margin: '0px'}} />
        <Button icon withLabel>Push Me</Button>
    </div> 
```
